# Local Image Folders

Large local image folders live here as `images/<source>/`.

The public Hugging Face dataset is `https://huggingface.co/datasets/ODanione/ai-bias-images`.

That dataset keeps AI-generated source folders at the dataset root, for example local `images/FLUX/` syncs to remote `FLUX/`.

License information for the shared AI-generated images is documented in the Hugging Face dataset card and repository metadata at `https://huggingface.co/datasets/ODanione/ai-bias-images`.

`images/AdobeStock/` and `images/GoogleSearch/` are not part of the Hugging Face dataset. Those folders contain third-party reference images from stock search and web image search, so they have separate copyright, licensing, and provider terms from the AI-generated corpus controlled by this project.
